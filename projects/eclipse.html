Eclipse / Echo

Eclipse is a messaging website that is currently in development. It's expected to release ~March 2024. It will have multiple features on release, such as real-time messaging, encrypted logins (encrypted messages will come when I learn how to make it), and much more.

Echo is a social media website similar to Twitter or Instagram. When it releases (around the same time as Eclipse), you'll be able to pick between ~10 theme options, and more will come soon.
